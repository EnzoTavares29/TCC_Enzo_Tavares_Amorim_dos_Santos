Application Overview for AI Analysis

Project: Junta Parafusada
Location: d:\Downloads\ufs\TCC\junta_parafusada

Purpose:
- Python desktop application for bolted joint dimensioning using a GUI.
- Uses Tkinter for the user interface.
- Implements two dimensioning methods: Shigley (combined stress) and COEMI (friction-based torque optimization).

Entry point:
- main.py
  - Creates Tk root window.
  - Sets title and geometry.
  - Instantiates AppController from controllers/app_controller.py.
  - Starts Tk main loop.

High-level workflow:
1. Joint Setup screen: collect number of bolts, joint width, joint height.
2. Bolt Layout screen: add bolts by x,y positions; validate bolts inside joint bounds; compute centroid.
3. Bolt Properties screen: select bolt diameter, series, material/class.
4. Load Definition screen: choose between centered load, in-plane eccentric, and out-of-plane eccentric loads.
5. Results screen: compare user-defined torque and COEMI recommended torque, show safety factors for bolts and the joint.

Calculation methods:
- Shigley method:
  - Calculates bolt preload Fp from material yield and area.
  - Calculates torque from preload and bolt diameter.
  - Computes equivalent von Mises stress and bolt safety factor FS_parafuso.
  - Computes joint friction safety factor FS_junta = (μ × ΣFp) / F.
- COEMI method:
  - Computes optimal torque T_optimal = F / (0.20 × μ × n_bolts × n_interfaces).
  - Recalculates preload and safety factors using optimal torque.
  - Aims for joint safety factor near 1.0.
